This is the files for the GiD SWGOH site. All of the images used on the site are located in the images folder, and each file is located within the main swgoh file. The CSS is used for every single webpage. The most modern version of Bootstrap is also used for the site. Browsers used must be HTML5 compatible for everything to work properly. The site is designed to work on mobile, tablet and desktop with a mobile first approach. The site is a work in progress and I will continue to add more information here when/if necessary!
